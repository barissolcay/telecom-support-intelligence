# Evaluation

Run `make evaluate` to regenerate the versioned synthetic dataset, train the baseline, and calculate component metrics. Reports contain the measured value, target gate, dataset version, and timestamp.

Classification uses Macro F1, Weighted F1, per-class precision/recall, language slices, calibration proxy, and latency. Priority evaluation emphasizes critical recall. Retrieval uses Recall@5, Precision@5, MRR, and nDCG@5. RAG checks expected source IDs, citations, refusals, and unsupported claims. Privacy evaluation measures recall and false positives over labeled fixtures.

No README value should be presented as achieved unless its report exists and was generated by the scripts in this repository.
